# Prodeman MD Web

This folder is ready to publish with GitHub Pages.

## Files
- `index.html`: main public page
- `.nojekyll`: avoids Jekyll processing on GitHub Pages

## Publish To GitHub Pages
1. Create a new repository on GitHub.
2. Open PowerShell in this folder.
3. Run:
   `git add .`
   `git commit -m "Initial web version"`
   `git remote add origin https://github.com/YOUR_USER/prodeman-md-web.git`
   `git push -u origin main`
4. In GitHub: Settings > Pages > Deploy from a branch > `main` > `/ (root)`.

## Result
Your public URL will look like:
`https://YOUR_USER.github.io/prodeman-md-web/`
